# Finance Management System

A web-based Finance Management System built with Django that helps users track and manage daily expenses efficiently. The application allows users to record transactions, monitor spending patterns, and analyze financial data through graphical reports.

## Features

- Add, update, and delete expense records
- Track daily and monthly expenses
- Filter expenses by date and month
- Visualize spending trends using line graphs
- User-friendly interface
- SQLite database integration

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap

## Project Structure

- Expense tracking and management
- Date-based filtering
- Monthly analytics and reports
- Graphical visualization of spending trends

## Requirements

Install all required packages using the `requirements.txt` file:

```bash
pip install -r requirements.txt
```


## Running the Project

Start the Django development server:
```bash
python manage.py runserver
```
Open your browser and visit:
```
http://127.0.0.1:8000/
```
